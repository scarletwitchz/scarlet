# Climatologia

## Tempo x Clima

### Tempo

* Estado **momentâneo** da atmosfera
* Meteorologia
* Ex.: "Hoje o tempo tá nublado"

### Clima

* Estudo do tempo da localidade ao longo de **30 anos**
* Climatologia
